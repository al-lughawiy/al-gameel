Relays

The Relay game is a game much like an actual relay race. A message is passed along from person to person, but with the added twist of conlangs. The first user provides a passage in their conlang along with the necessary lexicon and grammar to translate it, and the second user deciphers their message and translates it into their own conlang. They then provide the third user with the message and the needed information. This then repeats until the last person is reached.
